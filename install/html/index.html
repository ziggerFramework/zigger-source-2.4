퍼미션 체크

  • >'/data' 디렉토리에 쓰기 권한(707 이상)이 부여되어야 합니다.
  • >'/robots.txt' 파일에 쓰기 권한(707 이상)이 부여되어야 합니다.

PHP 체크

  • >PHP Core 버전이 5.5.0 이상이어야 합니다.
  • >'GD' extension 이 설치되어 있어야 합니다.
  • >'mbstring' extension 이 설치되어 있어야 합니다.
  • >'PDO' extension 이 설치되어 있어야 합니다.
  • >'curl' extension 이 설치되어 있어야 합니다.
  • >'OpenSSL' extension 이 설치되어 있어야 합니다.
